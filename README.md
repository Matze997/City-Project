# Commands

Test out sounds
```
/playsound [Sound] [Pitch = 1.0] [Volume = 1.0]
```

Command to test features
```
/test
```

Save current street network to disk 
```
/streetmapper save
```

Get streetmapping items
```
/streetmapper
```
